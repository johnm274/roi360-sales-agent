# Product Knowledge — ROI360 / PageFlex Features

## What Is ROI360?

ROI360 is a web-to-print platform built on PageFlex Storefront technology. It allows print businesses and agencies to create branded, self-service ordering portals for their clients. End users can select templates, customise them with their details, preview, approve, and order — all within a controlled, brand-safe environment.

**Tagline:** Brand and Marketing Control Software

## Core Platform Capabilities

### Template Management
- **Customisable templates** — professionally designed templates with editable fields (text, images, colours) and locked elements (logos, brand colours, layout)
- **Variable data** — templates can pull data from databases (names, addresses, pricing) for personalisation at scale
- **Multiple output formats** — print-ready PDFs, digital files, social media sizes
- **Template categories** — organise by type (business cards, leaflets, banners, social media, stationery)

### User Experience
- **Self-service portal** — users log in, browse templates, customise, and order without design skills
- **Brand-locked elements** — logos, colours, fonts, and layout cannot be changed by users, ensuring brand consistency
- **Live preview** — see exactly what the finished item looks like before ordering
- **Mobile-friendly** — accessible from any device

### Workflow & Approvals
- **Approval workflows** — configurable approval chains (e.g., manager must approve before production)
- **Business rules** — enforce ordering rules (minimum quantities, approved products per role/location)
- **User roles and permissions** — different access levels for different types of users (admin, approver, orderer)
- **Order tracking** — users can track their orders from placement to delivery

### Administration & Reporting
- **Admin dashboard** — see who's ordering what, template usage, order volumes, spend
- **User management** — add/remove users, assign to groups/locations, set permissions
- **Template management** — add, edit, archive templates; control which templates are available to which users
- **Reporting** — order history, usage analytics, spend tracking

### Print Integration
- **Direct-to-production** — orders flow directly into print production workflows
- **Multiple print partners** — can route orders to different print suppliers based on product type or location
- **Fulfilment options** — print-on-demand, stock management, direct mail, split shipping

### System Integration
- **SSO (Single Sign-On)** — SAML, Azure AD, Google for seamless login
- **API access** — integrate with existing systems (CRM, ERP, marketing platforms)
- **Data connections** — pull data from external databases for variable data templates

## What ROI360 Is NOT
- **Not a design tool** — it's not competing with Canva, InDesign, or Photoshop. Users don't design from scratch; they customise pre-built templates
- **Not a print MIS** — it doesn't replace your production management system; it feeds into it
- **Not a marketing automation platform** — it doesn't do email campaigns, social posting, or analytics (though it can integrate with tools that do)
- **Not a DAM (Digital Asset Management)** — it's focused on template-based production, not asset storage (though it handles assets within templates)
